# 🎵 AI 작곡 시스템 프로젝트 로드맵

## 📊 프로젝트 개요
- **프로젝트명**: AI Songwriting System
- **목표**: 다양한 장르의 음악을 AI로 생성하는 통합 시스템
- **기술 스택**: Python, FastAPI, React, Ableton Live, TensorFlow/PyTorch
- **대상 장르**: 록, 힙합, J-pop, 재즈, 클래식, 팝

## 🏆 전체 진행률: **30%** 완료

- **소요 시간**: 1시간 | **난이도**: ⭐⭐

#### 1.3 웹 프레임워크 및 데이터베이스 설정 ✅
- **소요 시간**: 2시간 | **난이도**: ⭐⭐⭐
- **완료 내용**:
  - FastAPI 서버 구축
  - SQLAlchemy ORM 설정
  - 6개 핵심 데이터 모델 구현:
    - User (사용자)
    - Composition (작곡)
    - Track (트랙)
    - MidiFile (MIDI 파일)
    - AbletonProject (Ableton 프로젝트)
    - GenerationParams (AI 생성 파라미터)
  - CORS 설정 (React 연동 준비)
  - API 엔드포인트 구현 (/health, /compositions)

#### 1.4 Ableton Live 개발 환경 구성 ✅

- **완료 내용**:
  - MIDI 통신 모듈 (python-rtmidi)
  - Ableton Live API 래퍼 클래스
  - Max for Live 디바이스 템플릿
  - 11개 새로운 API 엔드포인트:
    - `/ableton/status` - 연결 상태
    - `/ableton/connect` - Live 연결
    - `/ableton/project/*` - 프로젝트 관리
    - `/ableton/midi/*` - MIDI 통신
    - `/ableton/devices/*` - M4L 디바이스

## 🛠️ **기술 스택 상세**

### **백엔드**
- **언어**: Python 3.12+
- **프레임워크**: FastAPI
- **ORM**: SQLAlchemy
- **데이터베이스**: SQLite → PostgreSQL (배포시)
- **AI/ML**: TensorFlow, PyTorch, Transformers

### **프론트엔드**
- **언어**: TypeScript
- **프레임워크**: React 18+
- **상태관리**: Redux Toolkit
- **UI**: Material-UI, Tailwind CSS
- **빌드**: Vite

### **음악 처리**
- **MIDI**: python-rtmidi, mido, pretty_midi
- **오디오**: librosa, soundfile
- **음악이론**: music21
- **DAW 연동**: Ableton Live API, Max for Live

### **인프라**
- **컨테이너**: Docker, docker-compose
- **배포**: 추후 결정 (AWS/Azure/GCP)
- **버전관리**: Git, GitHub
