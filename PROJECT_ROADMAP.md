# 🎵 AI MIDI 생성 시스템 개발 로드맵

## 📋 프로젝트 개요
자연어 프롬프트를 통해 **트랙별, 송 파트별로 세분화된 MIDI 파일**을 생성하는 AI 시스템입니다. 복잡한 오디오 처리 대신 **고품질 MIDI 생성**에 집중하여 실용적이고 구현 가능한 솔루션을 제공합니다.

### 🎯 핵심 목표
- **모듈식 4마디 스니펫 생성**: 레고 블록식 음악 창작
- **3단계 장르 시스템**: 메인 장르 → 세부 장르 → 스타일 변형
- **아티스트 기반 생성**: 특정 아티스트 스타일 모방 + 국가별 변형
- **개인화된 데이터 관리**: 수동 검증 및 품질 관리 시스템
- **실용적 작곡 도구**: 개발자 본인의 송라이팅 및 음악 제작 지원
