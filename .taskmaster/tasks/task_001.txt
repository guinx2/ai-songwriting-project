Setup configuration management for API keys and model parameters. Create MusicParams dataclass for structured music parameter representation.

Pseudo-code:
```python
class LLMProvider(ABC):
    @abstractmethod
    def analyze_prompt(self, prompt: str) -> MusicParams

class ClaudeProvider(LLMProvider):
    def __init__(self, api_key: str)
    def analyze_prompt(self, prompt: str) -> MusicParams

class ExaoneProvider(LLMProvider):
    def __init__(self, api_key: str)
    def analyze_prompt(self, prompt: str) -> MusicParams
```
